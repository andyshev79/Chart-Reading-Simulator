# Chart Reading Simulator

Обучающий тренажёр технического анализа: **читай сигналы графика → оцени вероятность → узнай реальность → учись калибровке.**

Три ряда сигналов (графические фигуры · индикаторы и объёмы · свечные комбинации) выпадают на барабанах
и могут противоречить друг другу — как на реальном рынке. Игрок выбирает направление и **силу сигнала (%)**,
а в конце открывается истинная вероятность расклада. Продукт учит главному: **хорошее решение ≠ хороший исход** —
профессионал играет вероятности, а не результаты.

> ⚠️ Обучающая симуляция · не является инвестиционной рекомендацией.

**Языки:** EN / RU / UA / UZ.

## Группы инструментов (уровень сложности)
- **Низковолатильные** (FX-мажоры, защитные акции, облигации) — малые движения, паттерны надёжнее.
- **Трендовые** (акции, индексы, металлы, комоды) — средний баланс риск/доходность.
- **Высоковолатильные** (крипта, опционы) — большие движения, но паттерны врут чаще.

## Статус

Режим PRO (сумма позиции + мультипликатор) — на следующем этапе.
