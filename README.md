# Chart Reading Simulator

Обучающий тренажёр технического анализа: **читай сигналы графика → оцени вероятность → узнай реальность → учись калибровке.**

Три ряда сигналов (графические фигуры · индикаторы и объёмы · свечные комбинации) выпадают на лентах
и **могут противоречить друг другу** — как на реальном рынке. Игрок выбирает направление и **силу сигнала (%)**,
а в конце открывается истинная вероятность расклада. Продукт учит главному: **хорошее решение ≠ хороший исход** —
профессионал играет вероятности, а не результаты.

> ⚠️ **Обучающая симуляция · виртуальные средства · не является инвестиционной рекомендацией.**

**Live demo:** https://chart-reading-simulator.vercel.app

Языки интерфейса: **EN / RU / UA / UZ**.

---

## Как это работает

1. **Выбор группы инструмента** (уровень сложности) и длины сессии.
2. **Ленты сигналов** прокручиваются и выдают 3 сигнала — по одному из каждого ряда.
3. **Решение:** направление **↑ Вверх / ↓ Вниз / Пропустить** + слайдер **силы сигнала (%)**. Ставка фиксирована — $1000.
4. **Результат:** раскрывается истинная вероятность каждого сигнала и общая вероятность расклада, выносится **вердикт risk/reward**.
5. **Финал сессии:** качество решений, калибровка, звание, кривая динамики.

---

## Группы инструментов

Игрок выбирает группу в начале — это уровень сложности. Группа влияет на **амплитуду движения** и **надёжность паттернов**.

| Группа | Примеры | Амплитуда | Надёжность сигналов | Модификатор |
|--------|---------|-----------|---------------------|-------------|
| **Низковолатильные** | FX-мажоры, защитные акции, облигации | малая | выше (паттерны чище) | +7% |
| **Трендовые** | акции, индексы, металлы, комоды | средняя | базовая | 0% |
| **Высоковолатильные** | крипта, опционы | большая | ниже (много ложных пробоев) | −10% |

Урок: на крипте можно много заработать, но сигналы менее надёжны — риск/доходность требует осторожности.

## Ряды сигналов

- **Ряд F — Графические фигуры** (оранжевый): голова-плечи, двойная вершина/дно, треугольники, флаги, вымпел, клин, чашка с ручкой.
- **Ряд I — Индикаторы и объёмы** (синий): пересечения MA, золотой крест, RSI-дивергенция/перекупленность, MACD, пробой с объёмом и без, Боллинджер, отбой от уровня.
- **Ряд C — Свечные комбинации** (фиолетовый): поглощение, молот, падающая звезда, утренняя/вечерняя звезда, доджи, три солдата/вороны, харами.

Каждый сигнал — стилизованная SVG-иконка + название + описание. У каждого своя базовая вероятность отработки.

## Математика калибровки

**Вероятность сигнала** = базовая ± модификатор группы. Нейтральные (доджи) ≈ 50%.

**Комбинирование трёх сигналов** в общую вероятность движения вверх:
```
pUp_i = вероятность сигнала в сторону «вверх» (для «вниз» = 100 − prob)
raw    = Σ (pUp_i − 50)
pUp    = clamp(50 + raw × 0.7, 5, 95)
```
Согласованные сигналы усиливают друг друга; конфликтующие тянут итог к 50% (неопределённость → правильный ход часто «Пропустить»).

**Движение и P&L:**
```
исход вверх с вероятностью pUp; амплитуда ~ волатильность группы
pnl = dirSign × $1000 × (move% / 100)
```

**Вердикт risk/reward** (ядро обучения):

| Исход | Реальная вероятность направления | Вердикт |
|-------|----------------------------------|---------|
| Прибыль | высокая (≥58%) | ✅ Правильное решение |
| Прибыль | низкая (<45%) | ⚠️ Повезло (плохой risk/reward) |
| Убыток | высокая (≥58%) | 👍 Верное решение, не повезло |
| Убыток | низкая (<45%) | ❌ Ошибка |

Для «Пропустить»: правильно при раскладе ~50/50, «упущено» при сильном перевесе.

## Финал сессии

- **Звание** по качеству решений (не по удаче): Новичок → Читатель графика → Аналитик → Мастер калибровки.
- **Качество решений** — усреднённая оценка вердиктов.
- **Калибровка** — насколько оценки силы сигнала совпали с реальными вероятностями.
- **Динамика P&L** — кривая по раундам.

## Данные (patterns.json)

Весь контент — в `src/patterns.json`: `{ groups, rows, signals, groupMod }`.
Чтобы добавить сигнал — допиши объект в `signals`, код менять не нужно.

```jsonc
{
  "row": "F",              // F | I | C
  "id": "dbot",
  "dir": "up",             // up | down | neutral
  "base": 64,              // базовая вероятность отработки, %
  "icon": "doublebottom",  // тип стилизованной иконки
  "t": { "en": "...", "ru": "...", "ua": "...", "uz": "..." },  // название
  "d": { "en": "...", "ru": "...", "ua": "...", "uz": "..." }   // описание
}
```
